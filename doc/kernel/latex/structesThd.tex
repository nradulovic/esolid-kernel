\subsubsection{Detailed Description}
Thread structure. 

A thread structure is a data structure used by kernel to maintain information about a thread. Each thread requires its own I\-D structure and the structure is allocated in user memory space (R\-A\-M). The address of the threadâ€™s I\-D structure is provided to O\-S thread-\/related services.

Thread structure is used as thread I\-D and a thread is always referenced using this structure. \begin{DoxyParagraph}{Object class\-:}
